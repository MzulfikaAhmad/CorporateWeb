GITHUB UPLOAD STRUCTURE

Upload the contents of this folder into the ROOT of your existing corporate website repository.

Required location:
aboutme.html
assets/images/zul2.png

The page also uses these existing corporate website files:
assets/images/logo.svg
assets/css/style.css
assets/js/script.js

Important:
1. Do not upload the ZIP file directly as a website file. Extract it first.
2. Keep all file and folder names in lowercase exactly as shown.
3. In GitHub, choose Add file > Upload files, then upload aboutme.html and the assets folder.
4. Commit the changes and wait for GitHub Pages to redeploy.
5. Open: https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/aboutme.html
