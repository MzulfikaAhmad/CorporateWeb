PAGES
1. index.html - Home
2. about.html - About Us
3. activities.html - Activities & Events
4. news.html - News & Announcements
5. donation.html - Donation
6. location.html - Masjid Location
7. contact.html - Contact & Feedback

HOW TO OPEN
- Open index.html in a web browser.
- Keep the complete folder structure unchanged.

FEATURES MATCHING THE IMS458 RUBRIC
- HTML5 and one external CSS file: assets/css/style.css
- External JavaScript: assets/js/script.js
- Hierarchical dropdown navigation and responsive mobile menu
- Internal links between all pages
- External links to official PERKIM, MAIS, JAKIM, mosque, maps and social channels
- PDF link: assets/docs/PERKIM_Selangor_Community_Guide.pdf
- mailto email editor link
- Feedback form with browser storage, four-response progress, CSV export and print summary
- Image slider, flip cards, filters, tooltips, accordion, quiz, donation planner and tabs
- Responsive hover/focus effects and back-to-top control
- Disclaimer and source notes

IMPORTANT FEEDBACK EVIDENCE STEPS
1. Open contact.html in the same browser and device.
2. Ask at least four respondents to submit the form.
3. Take a screenshot of the response table.
4. Click Export CSV Evidence.
5. Insert the screenshot and respondent summary in the development report.
